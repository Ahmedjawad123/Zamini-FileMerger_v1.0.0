What is Zamini Converter?

Zamini Converter is a lightweight desktop tool that allows you to convert PDF files, images, and Office documents into one neat PDF ‚Äî without needing an internet connection.

Requirements

  • Microsoft Office installed for converting Office files (.docx, .xlsx, etc.)
  • Windows OS (built using Tkinter for Windows)

How It Works

  1. Click "Add Files" to select documents or images to convert.
  2. Reorder files using "Move Up" or "Move Down".
  3. Remove any with "Delete".
  4. Type a name for your final converted PDF.
  5. Click "Convert to PDF" and watch the progress bar finish.
  6. Save your converted PDF and you're done!
  7. Use "Reset" to start fresh anytime.

Who is it for?

  • Teachers & students who deal with multiple file types.
  • Office workers converting daily documents efficiently.
  • Print shops handling mixed file conversion requests.
  • Anyone looking for a simple offline document converter.
